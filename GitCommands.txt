### PORTARE I TUOI CAMBIAMENTI ATTUALI SU UN ALTRO BRANCH
 -> git stash push -m "Cambiamenti da portare su altro branch"
 -> git checkout nome-branch-destinazione
 -> git stash pop

### RINOMINA BRANCH REMOTO E LOCALE
 -> git branch -m {vecchio_nome} {nuovo_nome} 		 [Rinomina locale]
 -> git push origin --delete {vecchio_nome}   		 [Elimina remoto]
 -> git branch --set-upstream-to=origin/NUOVO_NOME   [Imposta nuovo nome branch remoto]
 -> git remote prune origin   						 [Rimuoverà tutti i riferimenti remoti che non esistono più]
