Linee guida per l'installazione di librerie da utilizzare nello sviluppo di un'app React-Native:

- expo-router per il routing
- State-management solution -> Zustand
- Secure key-value Storage -> MMKV
- Picker, dropdown and other generic components -> react-native-paper
- Date Picker -> react-native-date-picker
- Form -> react-hook-form
- Validazione form -> yup
- Visualizzazione cross-Platform -> react-native-safe-area-context
- gestione locale dei dati -> expo-sqlite
- internationalization -> i18next

Linee guida per la scrittura del codice fin dall'inizio nello sviluppo di un'app React-Native:
- usare un approccio di coding basato su dark/light mode nella creazione di pagine e componenti
- formattare le hardcoded string che servono come messaggi UI attraverso i file di configurazione di i18next (it, en)
